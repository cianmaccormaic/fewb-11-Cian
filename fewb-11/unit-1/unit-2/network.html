Common Network Protocols

Below is a list of three commonly used network protocols along with their descriptions:

Transmission Control Protocol (TCP)
TCP is a connection-oriented protocol that ensures reliable data transmission between devices over a network.
User Datagram Protocol (UDP)
UDP is a connectionless protocol that allows fast transmission of data but does not guarantee delivery or error checking.
Internet Protocol (IP)
IP is responsible for routing and addressing packets of data so they can travel across networks to their destination.
